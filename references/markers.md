# AI 腔常见标记与改法

这份参考给的是实用改法，不是“鉴定器”。

没有任何一个词能单独证明文本来自 AI。这里列的是一组高频风险点：当这些写法成串出现时，中文会更像模板和机器润色稿。

## 1. 滑词：句子很顺，但意思发虚

常见例子：

- `落到`
- `大面上`
- `这一层`
- `底色`
- `摆了出来`
- `接住了`
- `往前看`
- `更吃力的部分`

优先改法：

- `落到` -> `变成` / `体现在` / `写成`
- `大面上` -> `整体看` / `多数情况下`
- `接住了` -> `回应到了` / `处理到了` / `写进了病历`
- `底色` -> 直接写风险、起点、病历质量、结局

## 2. 连接词太勤，像讲义

高风险段首：

- `此外`
- `同时`
- `另一方面`
- `总的来看`
- `值得注意的是`
- `需要指出的是`
- `进一步来看`
- `从某种程度上`
- `在这一层面`

改法：

- 如果去掉连接词，句子还成立，就直接去掉。
- 让段首直接落在事实、问题、对象或结果上。

## 3. 评价写法本身，而不是讲事情

高风险写法：

- `这一点很关键`
- `这更像是在提醒我们`
- `信息很清楚`
- `边界也很明确`
- `最值得记住的是`

改法：

- 直接写结论
- 直接写限制
- 直接写数字意味着什么

## 4. 句式模板重复

常见模板：

- 先下一句总判断
- 再补一句解释
- 再补一句边界

或者连续几段都这样起手：

- `先说……`
- `接下来……`
- `再看……`

改法：

- 让不同段落有不同起手点：问题、对象、图、数字、结局
- 同一部分里，不要每段都用同一骨架

## 5. 动词太抽象，名词太多

高风险写法：

- `体现出`
- `构成了`
- `呈现出`
- `形成了一层变化`
- `提供了一种视角`

改法：

- 直接写动作：`增加了`、`降低了`、`漏掉了`、`纠正了`、`没有采纳`

## 6. 人味不靠夸张

目标不是网络腔，也不是营销腔。

保留：

- 平视
- 顺口
- 自然
- 克制

避免：

- 过度熟络
- 夸张感叹
- 强行制造亲密感
- 靠情绪词代替信息

## 快速自检

改完后问四个问题：

1. 这句话是不是在讲事实，而不是点评这段写作？
2. 这个词如果放进别的文章也通，是不是太空了？
3. 这一段是不是靠连接词在往前推？
4. 这段话听起来像人坐下来讲清楚一件事，还是像模板在出稿？
